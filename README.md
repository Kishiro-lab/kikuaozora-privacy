# 聞く青空 プライバシーポリシー

制定日: 2026年7月17日

「聞く青空」(以下「本アプリ」)は、個人開発者(Kishiro Lab)が提供する、青空文庫(aozora.gr.jp)の作品を検索・読み上げるためのAndroidアプリです。本アプリにおける利用者情報の取り扱いについて、以下のとおり定めます。

## 1. 本アプリが収集する情報

本アプリは、氏名・メールアドレス・位置情報などの個人を特定できる情報を収集しません。アカウント登録機能もありません。

本アプリが端末内に保存する情報は以下のみで、いずれも**端末内にのみ保存され、開発者や第三者のサーバーへ送信されることはありません**。

- ライブラリに追加した作品・話・読了位置・しおり
- 読み方辞書(単語の読み仮名)の設定内容
- 配色テーマ・文字サイズ等の表示設定

## 2. 通信について

本アプリが通信を行う先は、作品本文・作品情報を取得するための **青空文庫(aozora.gr.jp)** のみです。この通信には利用者を特定する情報は含まれません。

本アプリ自体はアクセス解析・広告SDK・クラッシュレポート等の第三者サービスを組み込んでいません(将来的に広告を導入する場合は、本ポリシーを改訂の上で明記します)。

## 3. 読み上げ機能について

読み上げには端末にインストールされているAndroid OS標準のテキスト読み上げ(TTS)エンジンを利用します。読み上げ対象の文章が外部の音声合成API等へ送信されることはありません。

## 4. バックアップ機能について

本アプリには、ライブラリ・設定・しおり・読み方辞書の内容をファイルに書き出し、任意の保存先(端末内ストレージ、Googleドライブ等)へ共有する機能があります。この書き出し・共有は利用者が明示的に操作した場合にのみ行われ、本アプリが自動的にどこかへアップロードすることはありません。書き出したファイルの保存先・共有範囲は、利用者が共有時に選択したアプリ・サービスの管理下に置かれます。

## 5. 端末の権限について

本アプリは、以下の目的でAndroidの権限を利用します。

- **ストレージへのアクセス**: バックアップファイルの書き出し・読み込みのため
- **通知**: 読み上げ中の再生・一時停止・スキップを通知やロック画面、イヤホンの操作ボタンから行うため(いわゆるメディア通知)

上記以外の目的で端末内のデータにアクセスすることはありません。

## 6. 対象年齢について

本アプリは特定の年齢層を対象としたものではなく、児童を主な対象とした設計・データ収集は行っていません。

## 7. 本ポリシーの変更について

本アプリの機能追加(広告導入等)に伴い本ポリシーの内容を変更する場合があります。重要な変更がある場合は、本ページの更新日を変更のうえ掲載します。
