# 聞く青空 プライバシーポリシー

制定日: 2026年7月17日
改訂日: 2026年7月17日(広告[Google AdMob]の導入に伴う改訂)

「聞く青空」(以下「本アプリ」)は、個人開発者(Kishiro Lab)が提供する、青空文庫(aozora.gr.jp)の作品を検索・読み上げるためのAndroidアプリです。本アプリにおける利用者情報の取り扱いについて、以下のとおり定めます。

## 1. 本アプリが収集する情報

本アプリは、氏名・メールアドレス・位置情報などの個人を特定できる情報を収集しません。アカウント登録機能もありません。

本アプリが端末内に保存する情報は以下のみで、いずれも**端末内にのみ保存され、開発者や第三者のサーバーへ送信されることはありません**。

- ライブラリに追加した作品・話・読了位置・しおり
- 読み方辞書(単語の読み仮名)の設定内容
- 配色テーマ・文字サイズ等の表示設定

## 2. 通信について

本アプリが通信を行う先は、作品本文・作品情報を取得するための **青空文庫(aozora.gr.jp)** と、下記「広告について」に記載の **Google AdMob** です。青空文庫との通信には利用者を特定する情報は含まれません。

本アプリはアクセス解析・クラッシュレポート等の第三者サービスは組み込んでいません。

## 3. 広告について

本アプリはライブラリ画面にバナー広告を表示するため、**Google AdMob**(Google LLC提供の広告配信サービス)を組み込んでいます。広告の表示・配信のため、広告SDKが端末の広告ID等の識別子や、おおよその位置情報(IPアドレスに基づくもの)、アプリの利用状況といった情報をGoogleへ送信する場合があります。これらはGoogleのプライバシーポリシー([https://policies.google.com/privacy](https://policies.google.com/privacy))に基づき取り扱われます。

EU/EEA・英国等、同意取得が法令上必要な地域からのアクセスに対しては、Googleが提供する同意管理の仕組み(User Messaging Platform)を通じて、パーソナライズ広告の可否について選択できる同意フォームを表示します。

## 4. 読み上げ機能について

読み上げには端末にインストールされているAndroid OS標準のテキスト読み上げ(TTS)エンジンを利用します。読み上げ対象の文章が外部の音声合成API等へ送信されることはありません。

## 5. バックアップ機能について

本アプリには、ライブラリ・設定・しおり・読み方辞書の内容をファイルに書き出し、任意の保存先(端末内ストレージ、Googleドライブ等)へ共有する機能があります。この書き出し・共有は利用者が明示的に操作した場合にのみ行われ、本アプリが自動的にどこかへアップロードすることはありません。書き出したファイルの保存先・共有範囲は、利用者が共有時に選択したアプリ・サービスの管理下に置かれます。

## 6. 端末の権限について

本アプリは、以下の目的でAndroidの権限を利用します。

- **ストレージへのアクセス**: バックアップファイルの書き出し・読み込みのため
- **通知**: 読み上げ中の再生・一時停止・スキップを通知やロック画面、イヤホンの操作ボタンから行うため(いわゆるメディア通知)

上記以外の目的で端末内のデータにアクセスすることはありません。

## 7. 対象年齢について

本アプリは特定の年齢層を対象としたものではなく、児童を主な対象とした設計・データ収集は行っていません。広告も児童を対象としない設定で配信されます。

## 8. 本ポリシーの変更について

本アプリの機能追加に伴い本ポリシーの内容を変更する場合があります。重要な変更がある場合は、本ページの更新日を変更のうえ掲載します。
